Description:


This is the final project of CS132. It consists of an e-commerce full-stack website with back-end and client end codes provided.

Installation:

Install dependencies: `npm istall`

To run the server on a localhost, make sure you have the following modules: express, muster (the command Nam install will install those automatically).
